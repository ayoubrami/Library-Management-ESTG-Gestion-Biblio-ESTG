* Developed by : [Ayoub Rami](https://github.com/ayoubrami)
## Technologies
 *  JavaFX
 *  CSS

 *  JFoenix

 *  [FontawesomeFX](https://bitbucket.org/Jerady/fontawesomefx) 'For icons'
